MULTI-USER SYNC

  

    

      How it works: Both users connect to a shared PocketBase server. Changes sync every 15 seconds automatically.


      Quick setup:

      1. Download PocketBase at pocketbase.io and run: pocketbase.exe serve

      2. Go to localhost:8090/_/ and create admin account

      3. Create collection named vault_data with one Plain text field: data

      4. Expose via ngrok: ngrok http --url=your-domain.ngrok-free.app 8090

      5. Paste URL below and share this HTML file with your second user
